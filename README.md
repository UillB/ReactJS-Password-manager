# ReactJS Password manager

React application that demonstrates ReactJS + Webpack + Babel;

## Requirements
* Git
* Node v6
* NPM v3
# To run the app locally:

```bash
git clone https://github.com/UillB/ReactJS-Password-manager.git
cd ReactJS-Password-manager
npm install
npm start
open http://localhost:8080
```
# Project structure

After creation, your project should like this:

```
ReactJS-Password-manager/
  README.md
  package.json
  package-lock.json
  .gitignore
  node_modules/
  webpack.config.js
  client/
    components/
        App.js
        Dashboard.js
        Register.js
        Siginin.js
    assets/
        images/
        main.scss
        variables.scss
    main.js
  public/
        build/
            bundle.js
        index.html
```
